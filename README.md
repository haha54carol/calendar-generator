# calendar-generator

This library is a simple generator of calendar object. :calendar:

## Installation
**This library work with `moment.js`.**

```
npm install calendar-generator
```
## Import
**ES6+, Babel, Webpack, etc.. environments**

```javascript
import { genCalendarObj } from 'calendar-generator'
```

## Example
```javascript
import { genCalendarObj } from 'calendar-generator'

const fromDate = moment().format('YYYY-MM-DD') // 2017-12-17
const toDate = moment(fromDate).add(1, 'months').format('YYYY-MM-DD') //2017-01-17
    
let cObj = genCalendarObj(fromDate, toDate)

```
